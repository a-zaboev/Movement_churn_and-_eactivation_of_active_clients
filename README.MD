### Функция для преобразования данных по продажам в статистические данные по движению активной базы, оттоку и реактивации
- функция принимает на вход сырые данные по продажам за неделю и регистрациям клиентов
- преобразование данных и алгоритм расчитывает асболютные цифры и доли клиентов:
  - оставшихся активными с прошлой недели
  - оттекших после активности на прошлой неделе
  - реактивировавшихся после отсутствия на прошлой неделе
  - новых клиентов (по описанному критерию)
  - АКБ начала и конца периода
- данные выводятся в таблицу, по которой можно получить полную информацию по активности клиентов
- критерии можно настроить под любую калибровку
